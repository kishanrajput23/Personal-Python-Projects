# Basic Web Scraping

A small example showing how to scrape data from web pages using Python libraries like `requests` and `BeautifulSoup`.

**Main script:** `basic web scraping.py`

How to run:

1. Install dependencies: `pip install requests beautifulsoup4`
2. Run: `python "basic web scraping.py"`

This project demonstrates extracting information from HTML and parsing elements of interest.
