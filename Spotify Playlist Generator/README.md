# Spotify Playlist Generator

A small script to create or manage Spotify playlists programmatically using the Spotify Web API.

**Main script:** `Spotify playlist generator.py`

How to run:

1. Register an app on the Spotify Developer Dashboard and obtain client credentials.
2. Install dependencies: `pip install spotipy`
3. Run: `python "Spotify playlist generator.py"` and provide required credentials.

This project shows how to authenticate with the Spotify API and create playlists by track IDs or search queries.
